# Ejemplo de Integración de Control de Versiones con SQL Server

## Descripción

Este repositorio fue creado como parte de una asignación académica. Su propósito es demostrar cómo se puede integrar el control de versiones con herramientas de administración de bases de datos.

Para este ejemplo se utilizó SQL Server Management Studio (SSMS) junto con Git, permitiendo versionar scripts SQL y vincular el repositorio a una plataforma remota como GitHub.

---

## Objetivo

El objetivo principal es mostrar de manera práctica:

- Cómo crear un repositorio desde SQL Server Management Studio
- Cómo vincularlo con un repositorio remoto en GitHub
- Cómo gestionar scripts SQL usando control de versiones

---

## Importante

Este repositorio es únicamente un **ejemplo educativo**.

- No representa un proyecto real en producción  
- No contiene una base de datos completa  
- Solo incluye scripts de ejemplo para demostrar el uso del control de versiones
